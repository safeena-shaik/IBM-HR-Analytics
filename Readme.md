# IBM HR Analytics Dashboard

An end-to-end Data Analytics project developed using **Python** and **Power BI** to analyze employee attrition and workforce trends.

---

## Project Overview

Employee attrition is one of the major challenges faced by organizations. This project analyzes the IBM HR Analytics dataset to identify factors influencing employee turnover and presents insights through an interactive Power BI dashboard.

The project covers the complete analytics workflow:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Dashboard Development
- Business Insights
- Recommendations

## Key Business Insights

- Total Employees: **1470**
- Employees Left: **237**
- Attrition Rate: **16.12%**
- Employees working overtime experienced higher attrition.
- Research & Development had the highest number of employees.
- Life Sciences was the most common education field.
- Most employees reported moderate to high job satisfaction.
